📊 Математические распределения

Интерактивная визуализация вероятностных распределений и их свойств

🔵 Нормальное распределение (распределение Гаусса) — одно из самых важных распределений в теории вероятностей. Часто встречается в природе и используется во многих областях науки.

⚙️ Параметры

Среднее (μ) 0
Стандартное отклонение (σ) 1

📈 Свойства

Среднее (μ)
0
Дисперсия (σ²)
1
Стандартное отклонение (σ)
1
Функция плотности вероятности (PDF):
f(x) = (1 / (σ√(2π))) × exp(-(x - μ)² / (2σ²))
📦 Равномерное распределение — распределение, при котором вероятность равномерна на интервале [a, b]. Используется в симуляциях и генерации случайных чисел.

⚙️ Параметры

Нижняя граница (a) 0
Верхняя граница (b) 10

📈 Свойства

Среднее (E[X])
5
Дисперсия (Var)
8.33
Минимум - Максимум
0 - 10
Функция плотности вероятности (PDF):
f(x) = 1 / (b - a), если a ≤ x ≤ b, иначе 0
⚡ Экспоненциальное распределение — описывает время между событиями в процессе Пуассона. Используется в теории надёжности и теории очередей.

⚙️ Параметры

Интенсивность (λ) 1

📈 Свойства

Среднее (E[X])
1
Дисперсия (Var)
1
Мода
0
Функция плотности вероятности (PDF):
f(x) = λ × exp(-λx), x ≥ 0
🎲 Биномиальное распределение — описывает число успехов в серии независимых испытаний Бернулли. Используется для моделирования дискретных случайных событий.

⚙️ Параметры

Число испытаний (n) 20
Вероятность успеха (p) 0.5

📈 Свойства

Среднее (E[X])
10
Дисперсия (Var)
5
Стандартное отклонение
2.236
Вероятностная функция (PMF):
P(X = k) = C(n,k) × p^k × (1-p)^(n-k)
📍 Распределение Пуассона — описывает число событий, происходящих в фиксированный интервал времени. Используется для моделирования редких событий.

⚙️ Параметры

Интенсивность (λ) 3

📈 Свойства

Среднее (E[X])
3
Дисперсия (Var)
3
Особенность
E[X] = Var(X) = λ
Вероятностная функция (PMF):
P(X = k) = (λ^k × e^(-λ)) / k!
